Upscaler

We are pleased to announce the launch of Upscaler, a plugin coming to the Unity Asset Store on October 20th, 2023! Upscaler provides support for NVIDIA DLSS in Unity Engine's Universal Render Pipeline and Built-in Render Pipeline.

Features

NVIDIA DLSS is supported for the following configurations.

  • Linux (Vulkan)
  • Windows (Vulkan, DirectX11, DirectX12)

Future Products

Conifer will release more plugins in the coming year, all introducing new features to either the Unreal or Unity game engine.
